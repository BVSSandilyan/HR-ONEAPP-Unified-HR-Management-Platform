# HR-ONEAPP-Unified-HR-Management-Platform
HR ONEAPP is a full-stack HR management system built with Flask, SQLAlchemy, and SQLite. It integrates attendance, payroll, leave management, tasks, notifications, scheduling, and video meetings into a unified role-based platform.

The platform supports three distinct roles — Admin, HR, and Employee — each with a tailored dashboard and permission boundary. Admins can start instant video meetings or fix a meeting to a future date, with a background scheduler that automatically activates it and notifies every employee and HR member the moment that date arrives, with no manual trigger required. A live notification system keeps everyone informed of meetings, task assignments, and leave decisions in real time, with quick-approve actions available directly from the notification bell.

Leave management is built around a transparent 20-day annual balance, with half-day requests counted proportionally. Employees can apply directly from their dashboard, with built-in safeguards: they cannot request leave before their joining date, cannot double-book already-approved dates, and receive a clear notice — rather than a silent rejection — if a request would exceed their remaining balance, since the system allows over-balance leave at an admin-defined salary deduction instead of blocking it outright. A dedicated salary-assignment widget on the dashboard automates this further, comparing an employee's approved monthly leave usage against a fair monthly allowance and calculating any deduction automatically, while preserving a clean separation from manually-entered payroll adjustments.

Every module — attendance, meetings, tasks, payroll, and leave — feeds into a unified, color-coded calendar so nothing exists in isolation. The interface itself is built around a formal, light corporate aesthetic with full dark-mode support, responsive layouts, and a real-time video meeting room with working mic, camera, and screen-share controls.

The result is a cohesive system where administrative actions — approving leave, starting a meeting, assigning salary — ripple correctly through attendance records, notifications, and payroll without requiring duplicate data entry, built for organizations that need their HR tooling to actually talk to itself.
